  <action>
Replace the entire index.html with a clean v2.0 version. This is a complete rewrite — the v1.0 prototype content (poem, statement, motif sections, gate, gift overlays) is NOT carried forward. Later phases will add content pages.

The new index.html must contain:

**Head:**
- Google Fonts preconnect + stylesheet link for Cormorant Garamond (italic 300-500) and Jost (200-400), same URL as current
- Viewport meta, charset, title "House Harlequin"

**CSS — Design Tokens (DESG-01):**
- `:root` custom properties for the full palette:
  - `--void: #050407` (deepest background)
  - `--lacquer: #0a0910` (card/panel background)
  - `--smoke: #141218` (elevated surface)
  - `--bone: #e8e0d0` (primary text)
  - `--champagne: #c9a96e` (accent/gold)
- Opacity variants for bone and champagne (same as existing: bone-70, bone-40, bone-20, bone-10, bone-05; champ-50, champ-25, champ-12, champ-06)
- `--ease-brand: cubic-bezier(0.16, 1, 0.3, 1)` as a named custom property
- `--font-display: 'Cormorant Garamond', serif`
- `--font-utility: 'Jost', sans-serif`

**CSS — Reset and Base (DESG-01, DESG-02):**
- Universal box-sizing reset (same as current)
- `body`: background var(--void), color var(--bone), font-family var(--font-utility), font-weight 300, antialiased rendering, overflow-x hidden
- `::selection`: background var(--champagne), color var(--void)

**CSS — Film Grain Overlay (DESG-04):**
- `body::after` with the same SVG noise texture as current, fixed position, pointer-events none, z-index 9990, opacity 0.025, background-size 200px

**CSS — Motion System (DESG-06):**
- `@keyframes rise`: to { opacity: 1; transform: translateY(0) }
- All transitions throughout MUST use `var(--ease-brand)` — never write the cubic-bezier longhand inline
- `@media (prefers-reduced-motion: reduce)`: disable all animation/transition durations, force visible elements to opacity 1 and transform none

**CSS — Scroll Reveal Utility (DESG-05):**
- `.observe` class: opacity 0, transform translateY(24px), transition opacity 1.4s var(--ease-brand), transform 1.4s var(--ease-brand)
- `.observe.visible`: opacity 1, transform translateY(0)
- Stagger variant: `.observe[data-delay="1"]` through `[data-delay="3"]` adding transition-delay of 0.15s, 0.3s, 0.45s

**CSS — Typography Utilities (DESG-02):**
- `.text-display`: font-family var(--font-display), font-weight 300, font-style italic
- `.text-utility`: font-family var(--font-utility), font-weight 200, font-size 8px, letter-spacing 0.5em, text-transform uppercase
- `.text-utility-light`: same as text-utility but color var(--bone-10)

**CSS — Lozenge Motif (DESG-03):**
- `.lozenge-divider`: a thin vertical line (1px wide, ~48px tall) with a 6px diamond shape centered on it, using champagne-25 color, margin auto for centering. Implemented as a single element with `::before` (line) and `::after` (diamond via rotated square). This is a structural divider, not decoration.
- `.lozenge-accent`: a small inline diamond mark (8px rotated square) for use as a bullet or inline accent, champagne-25 border, no fill

**HTML — Demo Layout:**
The body should contain a minimal demo page that exercises every token — NOT the v1.0 content. Structure:

1. A full-viewport hero section with void background, centered display text: "House Harlequin" in Cormorant Garamond italic (clamp 32px-5vw-56px), color bone-40, with a `.lozenge-divider` below it and a utility-text tagline "The art of noticing" in champagne-25 below that. All three elements have `.observe` class with staggered data-delay.

2. A second section (min-height 60vh, background lacquer) with a centered block of two paragraphs in Cormorant Garamond (clamp 18px-2.5vw-24px, line-height 2, color bone-20) demonstrating the display font. Separated by a `.lozenge-divider`. These paragraphs have `.observe`.

3. A third section (min-height 40vh, background void) with a utility-text label and a simple champagne-bordered box (smoke background) demonstrating the smoke and lacquer surfaces. Has `.observe`.

Each section should use padding clamp(4rem, 10vh, 10rem) vertically.

**JS — IntersectionObserver (DESG-05):**
- Same pattern as current: create observer with threshold 0.2, query all `.observe` elements, add `.visible` on intersection
- Keep it minimal — no gate logic, no overlays, no password. Just the observer.

**Important constraints:**
- Do NOT carry forward any v1.0 content (poem lines, brand statement, motif section, glimpse grid, invitation, gate, gift overlay, footer)
- Do NOT include the video hero — that is Phase 8
- The demo content exists solely to prove the design system works; it will be replaced in Phase 8
- Keep the single-file architecture (CSS in style tag, JS in script tag)
- Use var(--ease-brand) everywhere, never the raw cubic-bezier value
