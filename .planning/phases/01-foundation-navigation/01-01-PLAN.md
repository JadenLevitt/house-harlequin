# Phase 1 Plan 1: Foundation and Navigation

## Objective

Set up the Vite 7 project scaffold, create the single-page HTML structure with embedded CSS and JS, establish the full design token system (palette, typography, spacing), implement the SVG lozenge motif, responsive layout skeleton with section placeholders, fixed navigation bar with scroll-aware blur, and smooth-scroll linking.

## Context

- @.planning/research/STACK-HOUSE-HARLEQUIN.md (stack decisions: Vite 7, vite-plugin-singlefile, vanilla)
- @.planning/research/ARCHITECTURE-HOUSE-HARLEQUIN.md (CSS @layer architecture, component boundaries)
- @.planning/REQUIREMENTS.md (FOUND-01 through NAV-06)
- @.planning/PROJECT.md (core value: heavy black envelope)

## Tasks

### Task 1: Project Scaffold
type="auto"

Create the Vite 7 vanilla project with package.json, vite.config.js (with vite-plugin-singlefile), and the base index.html shell.

**Done criteria:**
- package.json exists with vite and vite-plugin-singlefile as devDependencies
- vite.config.js configures singlefile plugin
- index.html has doctype, meta viewport, font preconnect links
- `npm run dev` starts without errors

### Task 2: Design Tokens and CSS Foundation
type="auto"

Implement the full CSS @layer architecture with design tokens (palette, typography, spacing, timing), reset layer, and responsive foundation.

**Done criteria:**
- CSS custom properties for all 7 palette colors defined
- Typography system: Cormorant Garamond italic for display, Jost 300 caps for utility
- Google Fonts loaded with preconnect and display=swap
- @layer ordering: reset, tokens, typography, layout, components, animations, utilities
- prefers-reduced-motion media query in utilities layer

### Task 3: SVG Lozenge Motif System
type="auto"

Create reusable inline SVG definitions for the lozenge (elongated diamond) motif used as dividers, accents, and decorative elements.

**Done criteria:**
- SVG defs block with reusable lozenge symbol
- Lozenge usable as divider between sections
- Lozenge rendered in champagne color
- Multiple size variants available via CSS

### Task 4: Page Structure and Section Placeholders
type="auto"

Build the semantic HTML structure with all section placeholders (hero, manifesto, house-code, collection, world, enquire, footer) and responsive layout grid.

**Done criteria:**
- All sections exist as semantic HTML elements with proper IDs
- Desktop multi-column grid containers in place
- Single-column collapse on mobile viewport
- Generous vertical rhythm (minimum 16vh padding between sections)
- Section placeholder content visible for scroll testing

### Task 5: Fixed Navigation Bar
type="auto"

Implement the fixed dark nav bar with brand name, lozenge divider, navigation links, scroll-aware backdrop blur, and smooth-scroll behavior.

**Done criteria:**
- Fixed nav bar floats above all content
- Left: "HOUSE HARLEQUIN" in Jost, all caps, 11px, letter-spacing 0.25em, champagne
- Right: COLLECTION / THE HOUSE / ENQUIRE links with same type treatment
- Center: small lozenge SVG divider in champagne
- On scroll past hero: backdrop-filter blur(12px) + slight darkening
- Clicking nav link smooth-scrolls to corresponding section
- Nav bar does not overlap section content (proper body padding/margin)

## Verification

1. Page loads with Cormorant Garamond and Jost fonts visible
2. Fixed nav bar displays brand name, lozenge divider, and three links
3. Clicking a nav link smooth-scrolls to the corresponding section placeholder
4. Page layout collapses from multi-column to single-column on mobile viewport
5. Nav bar gains backdrop blur when scrolled past the first viewport height

## Success Criteria

- All FOUND-01 through FOUND-07 requirements addressed
- All NAV-01 through NAV-06 requirements addressed
- `npm run dev` serves the page without errors
- `npm run build` produces a single HTML file
