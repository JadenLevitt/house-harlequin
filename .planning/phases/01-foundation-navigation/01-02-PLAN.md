# Phase 1 Plan 2: Navigation Polish and Verification

## Objective

Verify the existing navigation bar meets all NAV-01 through NAV-06 requirements, then enhance it with refined hover interactions (letter-spacing expansion instead of opacity fade), proper wordmark treatment for the brand name, and polished lozenge divider positioning.

## Context

- @.planning/phases/01-foundation-navigation/01-01-SUMMARY.md (what was already built)
- @.planning/REQUIREMENTS.md (NAV-01 through NAV-06)
- @.planning/PROJECT.md (core value: heavy black envelope)

## Tasks

### Task 1: Audit Existing Nav Against Requirements
type="auto"

Review the current navigation implementation in index.html and verify each NAV requirement is met. Document any gaps or needed refinements.

**Done criteria:**
- All NAV-01 through NAV-06 requirements verified present in index.html
- Any gaps or refinements identified

### Task 2: Enhanced Nav Link Hover with Letter-Spacing
type="auto"

Replace the basic opacity hover effect on nav links with a refined letter-spacing expansion animation. This creates a more luxurious, intentional hover state that aligns with the brand's precision aesthetic.

**Done criteria:**
- Nav links expand letter-spacing on hover (from 0.25em to ~0.35em)
- Transition uses the luxury easing curve (cubic-bezier(0.16, 1, 0.3, 1))
- Opacity hover removed or relegated to secondary
- Hover feels deliberate and unhurried

### Task 3: Wordmark Brand Treatment
type="auto"

Refine the "HOUSE HARLEQUIN" brand name in the nav with proper wordmark styling -- subtle differentiation from the nav links to establish hierarchy.

**Done criteria:**
- Brand name has distinct visual treatment from nav links
- Hover state on brand name is subtle (letter-spacing or opacity)
- Brand name anchors to #hero section (already implemented)

### Task 4: Lozenge Divider Polish
type="auto"

Verify the center lozenge divider is properly sized, positioned, and colored. Ensure it doesn't shift on scroll state change and maintains visual harmony.

**Done criteria:**
- Lozenge SVG renders at correct size in nav center
- Position is stable across scroll states (no layout shift)
- Color is champagne, consistent with brand tokens

### Task 5: Visual Verification
type="checkpoint:human-verify"

Verify the navigation bar looks and behaves correctly in the browser.

**Verification steps:**
1. Open http://localhost:5173
2. Nav bar visible at top with brand name, lozenge, and links
3. Hover over nav links to see letter-spacing expansion
4. Scroll down to verify backdrop blur activates
5. Click nav links to verify smooth scroll

## Verification

1. Nav bar displays correctly with all elements (brand, lozenge, links)
2. Hover interactions use letter-spacing expansion
3. Scroll-aware backdrop blur works
4. Smooth scroll functions for all nav links
5. No layout shift on hover or scroll state change

## Success Criteria

- All NAV-01 through NAV-06 requirements verified and polished
- Letter-spacing hover replaces opacity hover on nav links
- Brand wordmark has distinct treatment
- Lozenge divider properly positioned and stable
